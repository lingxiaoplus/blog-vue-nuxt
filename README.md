
# 个人博客系统前端

使用nuxt重构的blog-vue项目，将页面静态化。更好的seo。




## 技术栈

主要采用了：

1. vuetify
2. axios
3. nuxt
4. ElementUI
5. vue-echarts
6. vue-mdEditor
7. vue-router

还有其他一些琐碎的技术，就不在这里列举了。

## 快速运行



克隆项目到本地，进入根目录中运行下面的命令：

```shell
# 安装依赖
npm install

# 在 localhost:9090 启动项目
npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
